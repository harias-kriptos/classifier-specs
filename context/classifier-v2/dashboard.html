⭐ Arquitectura central

State Machine compartido

Una sola tabla DynamoDB orquesta el flujo completo. Las 3 fases leen y escriben aquí. DDB Stream + EventBridge Pipes propagan los cambios.

Table: classifier-cycles-state
PK: enterprise_id
SK: CYCLE# / STATION# / REQUEST#
Estados: scanning → stations_complete → confirmed → phase2_collecting → complete
